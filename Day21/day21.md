# Day 21 - Digital Privacy Dashboard

## Objective

Generate a Digital Privacy Dashboard using Claude AI from the provided prompt and analyze the results.

---

## Steps Performed

1. Opened Claude.
2. Set effort level (Low/Medium/High).
3. Started a new conversation.
4. Pasted the Digital Privacy Dashboard prompt.
5. Reviewed the sample dataset.
6. Generated the complete HTML dashboard.
7. Saved the HTML file.
8. Opened it in a browser.
9. Tested all dashboard interactions.
10. Captured screenshots.
11. Uploaded all files to GitHub.

---
