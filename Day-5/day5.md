## Prompt A

### Prompt
Create a 30-day learning roadmap.

Include:
- Weekly milestones
- Daily tasks
- Resources
- Projects
- Final outcome

Make it practical and beginner-friendly.



## Prompt B

### Prompt
Create a 30-day learning roadmap.

Context:
- Current Situation: [Student/Professional/Freelancer]
- Current Skills: [Add Skills]
- Goal: [Target Goal]
- Available Time: [Hours per Day]
- Experience Level: [Beginner/Intermediate]
- Preferred Learning Style: [Videos/Projects/Reading]

Include:
- Weekly milestones
- Daily tasks
- Resources
- Projects
- Final outcome

Make it practical and beginner-friendly.



---

## Comparison

- Prompt B produced more personalized responses.
- Prompt B provided recommendations aligned with my goals.
- Prompt A was generic and less detailed.

---

## Key Learnings

1. Context significantly improves AI output quality.
2. Personalized prompts generate more relevant responses.
3. Detailed background information helps AI understand requirements better.
4. Context Engineering is essential for obtaining accurate and useful responses.
