# Day 2 - Prompt Engineering Challenge

## Objective
Compared outputs generated from a lazy prompt and an engineered prompt using Claude.

## Work Done
1. Opened Claude and started a new chat.
2. Entered a lazy prompt:
   "Create an image explaining Prompt Engineering"
3. Saved the generated output.
4. Started a new Claude chat.
5. Entered the engineered prompt.
6. Generated and compared both outputs.

## Observations

### Lazy Prompt Output
- Generic explanation
- Limited structure
- Fewer examples
- Basic image design

### Engineered Prompt Output
- Better organization
- Clear explanations
- More practical examples
- Improved visual hierarchy
- Professional image layout

## Key Learnings
- Detailed prompts improve output quality.
- Context and instructions lead to better responses.
- Prompt engineering significantly impacts AI-generated content.

## Prompts Used

### Lazy Prompt
Create an image explaining Prompt Engineering

You are an AI educator teaching complete beginners.

Include:
