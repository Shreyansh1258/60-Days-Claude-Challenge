# Day 17 – Vehicle Cost Analysis Dashboard

## Objective
Analyze long-term vehicle ownership costs using Gasoline and E85 fuel.

## Key Findings
- E85 saved $2,393 over 5 years.
- Break-even achieved at 28,540 miles.
- CO₂ emissions reduced by 1.24 tons.

## Dashboard Features
- KPI Cards
- Fuel Comparison
- Maintenance Analysis
- Environmental Impact
- SVG Charts

## Learnings
- Cost modeling
- Fuel economics
- Data visualization
- HTML dashboard development
